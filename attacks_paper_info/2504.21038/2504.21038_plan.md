#### Functions

1. **Static Template Attack**
   - **Purpose**: Execute a static template attack using predefined prefill templates.
   - **Inputs**: `model`, `query`, `category`
   - **Outputs**: `response`
   - **Side Effects**: None
   - **Assumptions**: Prefill templates are correctly defined for each category.
   
   ```python
   def static_template_attack(model, query, category):
       """
       Perform a static template attack using a predefined prefill template.
       
       Args:
           model: The victim model to attack.
           query: The harmful query to use.
           category: The attack category to use for the prefill.
       
       Returns:
           The response from the model.
       """
       prefill = PREFILL_TEMPLATES[category]
       response = model.generate_response(query, prefill)
       return response
   ```

   ```python
   def adaptive_attack(model, query, initial_prefill):
   ```

3. **Generate Response**
   - **Purpose**: Generate a response from the model using a query and prefill.
   - **Inputs**: `query`, `prefill`
   - **Outputs**: `response`
   - **Side Effects**: Calls the model's API.
   
   ```python
   def generate_response(query, prefill):
       """
       Generate a response from the model using a query and prefill.
       
       Args:
           query: The user query.
   ```

   - **Outputs**: `bool`
   - **Side Effects**: None
