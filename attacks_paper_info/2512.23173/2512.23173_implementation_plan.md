## 2. Attack Overview
- **Summary**: EquaCode is a multi-strategy jailbreak attack that combines mathematical equation solving and code completion. It transforms a malicious query into a mathematical equation ($Subject + Tool + Steps = MaliciousQuery$) and then embeds this equation into a Python `Solver` class context. The LLM is tricked into "solving" for the unknown "Steps" variable by writing code, effectively generating the malicious content under the guise of completing a code task.
- **Core Innovation**: The synergistic combination of "equationization" (transforming intent into algebraic form) and "code encapsulation" (wrapping the problem in a programming context) to bypass safety filters that focus on natural language or single domains.
