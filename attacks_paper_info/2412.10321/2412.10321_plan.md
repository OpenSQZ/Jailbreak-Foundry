### Prefix Generation and Selection
#### Candidate Prefix Generation
1. Use uncensored models to generate prefixes via guided decoding:

```python
def generate_candidate_prefixes(model, tokenizer, request):
    inputs = tokenizer(request, return_tensors="pt")
    outputs = model.generate(**inputs, do_sample=True, temperature=TEMPERATURE, max_length=50)
    candidate_prefixes = [tokenizer.decode(output, skip_special_tokens=True) for output in outputs]
    return candidate_prefixes
```

#### Preprocessing Candidate Prefixes
2. Apply augmentation and filtering, i.e., remove refusal-starters and duplicates:

```python
def preprocess_prefixes(candidate_prefixes):
    augmented_prefixes = set()
    for prefix in candidate_prefixes:
        if not prefix.lower().startswith(('i cannot', 'refuse')):
            augmented_prefixes.add(prefix.replace("Here is", "Here's"))
    return list(augmented_prefixes)
```

#### Evaluation with Two Criteria
3. Calculate Negative Log-Likelihood (NLL) for each prefix using victim models:

```python
def compute_nll(victim_model, tokenizer, prefix):
    inputs = tokenizer(prefix, return_tensors="pt")
    with torch.no_grad():
        outputs = victim_model(**inputs, labels=inputs["input_ids"])
    return outputs.loss.item()
```

4. Calculate Prefilled ASR via manual prompt attack:

```python
def compute_prefilled_asr(victim_model, tokenizer, prefix, request):
    # Construct the prompt and prefill with the prefix
    prompt = f"{request} {prefix}"
    inputs = tokenizer(prompt, return_tensors="pt")
    completions = []
    for _ in range(NUM_RESPONSES):
        outputs = victim_model.generate(**inputs, do_sample=True, temperature=TEMPERATURE, max_length=50)
        completions.append(tokenizer.decode(outputs[0], skip_special_tokens=True))
    harmful_count = sum([is_harmful_response(comp) for comp in completions])
    return harmful_count / NUM_RESPONSES
```
* The function `is_harmful_response()` identifies if a response qualifies as "harmful" using the nuanced evaluation criteria.

#### Prefix Selection Pipeline
5. Select top k prefixes based on combined criterion scores:

```python
def select_prefixes(victim_model, tokenizer, request, candidate_prefixes, k=2):
    candidates = []
    for prefix in candidate_prefixes:
        nll = compute_nll(victim_model, tokenizer, prefix)
        prefilled_asr = compute_prefilled_asr(victim_model, tokenizer, prefix, request)
        combined_score = PREFILLING_ASR_WEIGHT * np.log(prefilled_asr) - nll
        candidates.append((prefix, combined_score))
    # Sort by combined score, select top-k
    candidates.sort(key=lambda x: x[1], reverse=True)
    return [candidate[0] for candidate in candidates[:k]]
```

### Experiment Execution
6. Execute jailbreak attacks with selected prefixes and target victim models:

```python
def exec_jailbreak(victim_model, tokenizer, request, selected_prefixes, suffix_length=40):
    for prefix in selected_prefixes:
        attack_prompt = f"{prefix} {'!' * suffix_length}"
        inputs = tokenizer(attack_prompt, return_tensors="pt")
        outputs = victim_model.generate(**inputs, do_sample=True, temperature=TEMPERATURE, max_length=50)
        response = tokenizer.decode(outputs[0], skip_special_tokens=True)
        evaluate_harmfulness(response)
```
