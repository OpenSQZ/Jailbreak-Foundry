## 2. Attack Overview
**JAIL-CON** is an iterative attack framework that exploits "task concurrency" in LLMs. It combines a harmful task with a benign auxiliary task at the word level, interleaving them (e.g., `Harm {Benign} Harm {Benign}`). The attack uses two modes: **CVT** (Concurrency with Valid Task), where the model answers both tasks, and **CIT** (Concurrency with Idle Task), where the model answers the harmful task while outputting "idle" tokens (spaces) for the auxiliary task. The attack forces the model to process divergent intents simultaneously, bypassing guardrails.

## 5. Data Flow and Control Flow
1. **Initialization**: Load `jbb-benign` dataset (once).
2. **Generate Attack**:
    - Input: `goal` (Harmful prompt), `target` (Harmful target start), `attempt_index`.
    - Determine Mode: If `random`, use `attempt_index % 2` to toggle CVT/CIT (or hash for randomness).
    - Select Aux Task: Use `hash(attempt_index)` to pick a random benign task from loaded dataset.
    - Combine:
        - `combined_goal` = Interleave(`goal`, `aux_goal`)
        - `combined_target` = Interleave(`target`, `aux_target` [if CVT] else `spaces`)
    - Construct Prompt:
        - Select Template (Auto1 for CIT, Auto2 for CVT).
        - Replace separators in template.
        - `user_message` = `combined_goal` + `\nAnswer: ` + `combined_target`.
    - Output: Formatted string with SYSTEM and USER parts.

## 7. Detailed Implementation Instructions

1.  **Imports & Class Definition**:
    - Import `ModernBaseAttack`, `AttackParameter`.
    - Import `JBBBehaviorsLoader`.
    - Define `JailConAttack` inheriting from `ModernBaseAttack`.
    - Define `NAME = "JAIL-CON"`, `PAPER = "2510.21189"`.

2.  **Parameters (`PARAMETERS` dict)**:
    - `mode`: `AttackParameter(name="mode", type=str, default="random", choices=["cvt", "cit", "random"])`
    - `separator`: `AttackParameter(name="separator", type=str, default="{}")`
    - `seed`: `AttackParameter(name="seed", type=int, default=42)`

3.  **Initialization (`__init__`)**:
    - Call `super().__init__`.
    - Initialize `self.aux_dataset = None`.
    - Load the dataset: `loader = JBBBehaviorsLoader(); self.aux_dataset = loader.load("jbb-benign")`.
    - Store templates as class constants (copied from `Parallel_Prompts.py`):
        - `PROMPT_BEGIN`: The rules (Rule 1-4).
        - `TEMPLATE_CIT` (Auto1): `PROMPT_BEGIN` + CIT specific instructions + Example.
        - `TEMPLATE_CVT` (Auto2): `PROMPT_BEGIN` + CVT specific instructions + Example.

4.  **Helper Method: `combine_sentences(self, s1, s2, separator)`**:
    - Implement logic from `Parallel_QA.py`:
        - Split `s1`, `s2` by whitespace (`.split()`).
        - `max_len = max(len(w1), len(w2))`.
        - Pad shorter list with `" "` (string containing a space) to `max_len`.
        - Interleave: `f"{w1} {left}{w2}{right}"` for each pair.
        - Join with spaces.
    - `left` = `separator[0]`, `right` = `separator[1]`.
    - Return combined string.

5.  **Method: `generate_attack`**:
    - Args: `prompt` (goal), `target` (optional, default "Sure, here is"), `attempt_index` (default 1), `**kwargs`.
    - **Mode Selection**:
        - If `self.mode == "random"`: `current_mode = "cit" if attempt_index % 2 == 0 else "cvt"` (Alternate).
        - Else: `current_mode = self.mode`.
    - **Aux Task Selection**:
        - Use `random.Random(self.seed + attempt_index)` to pick an index `idx` from `0` to `len(self.aux_dataset) - 1`.
        - Get `aux_task = self.aux_dataset[idx]`.
        - `aux_goal = aux_task.goal`
        - `aux_target = aux_task.target`
    - **Combination**:
        - `combined_goal = self.combine_sentences(prompt, aux_goal, separator)`
        - If `current_mode == "cvt"`:
            - `combined_target = self.combine_sentences(target, aux_target, separator)`
        - Else (`cit`):
            - `combined_target = self.combine_sentences(target, "", separator)` (Empty string for s2 means it will be padded with spaces).
            - Note: `combine_sentences` implementation pads with `" "` if list is shorter. Passing `""` to split gives `[]`. So `words2` will be `[]`, padded to `len(words1)` with `" "`.
    - **Template Selection**:
        - `template = self.TEMPLATE_CVT if current_mode == "cvt" else self.TEMPLATE_CIT`
        - Replace `{` and `}` in template with `separator` chars (if separator is different).
    - **Formatting**:
        - Construct final string:
          ```
          SYSTEM: {template}
          USER: {combined_goal}
          Answer: {combined_target}
          ```
          (Note: Ensure exact formatting matches `Parallel_QA.py` construction: `content = prompt + '\nAnswer: ' + combine_sentences(...)`. `Parallel_QA` puts the `content` in `USER` role).

6.  **Important Note on Extraction**:

    - Evaluation requires extracting the words outside the separators.
